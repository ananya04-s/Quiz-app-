# Simple Quiz App

## Overview

Simple Quiz App is an Android application developed using Kotlin that allows users to answer a multiple-choice question and instantly receive feedback on their selection. The application demonstrates fundamental Android development concepts such as user interface design, event handling, input validation, and dynamic UI updates.



---

## Features

* Interactive quiz interface
* Multiple-choice question using RadioButtons
* Answer validation on button click
* Instant result display
* Correct and incorrect answer feedback
* Input validation for unanswered submissions
* Disabled options after submission
* Simple and user-friendly design
* Reset functionality for retaking the quiz

---

## How It Works

1. User opens the application.
2. A quiz question and answer choices are displayed.
3. User selects one answer.
4. User clicks the Submit button.
5. The application validates the selected answer.
6. The result is displayed as:

   * Correct
   * Incorrect
7. Answer options are disabled after submission to prevent multiple attempts.
